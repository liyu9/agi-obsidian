# 求知心态：产品经理的持续学习

## 核心内涵

产品经理需要保持饥渴的学习心态，每隔三个月或半年回头看自己是否有进步。产品经理的每一步成长都来源于实践的经验与日常生活、工作中的积累。

## 产品经理知识矩阵

产品经理知识涉猎广泛，从市场、用户、产品、设计、运营、技术、商业，到心理学、美学，包罗万象。

### 知识矩阵图

```
        市场/商业
           ↑
         运营
        ↗   ↘
      设计 ← 产品 → 技术
        ↖   ↙
         心理学
           ↓
         美学/其他
```

## 求知心态的表现

### 日常积累
- 在地铁上观察用户使用什么产品、怎么使用
- 每天工作、生活的十几个小时中不断积累
- 将积累应用到产品上，通过数据和反馈验证对错

## 学习方法

> "汝果欲学诗，工夫在诗外。"——陆游

产品经理应当：
- 看工具书（用户研究、交互设计、视觉设计）
- 看经典市场营销书籍
- 了解互联网新技术、新方向

## 关键习惯

### 三个月/半年自检法
每过三个月或半年回头看看自己是否有进步，如果没有，应该感到恐慌。

### 面试考察
互联网公司在面试产品经理时，常会：
- 看候选人手机里有没有新奇的App
- 问最近有没有遇到好玩的产品

## 成长阶段

- 工作前三年养成强烈的求知心态至关重要
- 良好的学习习惯从学生时代培养
- 终身学习是产品经理应追求的目标
