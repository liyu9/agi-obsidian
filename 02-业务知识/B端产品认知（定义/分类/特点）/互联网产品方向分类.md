# 互联网产品方向分类

## 1.2 互联网与科技领域的产品方向

根据产品的属性和目标，互联网公司习惯将产品分为：
- C端产品
- B端产品
- 数据与策略产品
- 商业变现产品
- AI产品

在乙方软件公司，目标客户主要以企业和政务为主，产品分类没有互联网公司这样细致，一般统称为B端产品。

### 1.2.1 C端产品（2C产品）

**定义**
- C端产品也叫2C(to Customer)产品
- 面向终端用户或消费者的产品
- 往往承担流量获取和转化的重任

**C端产品特点**

| 特点 | 说明 |
|------|------|
| 用户是个体 | 使用C端产品的是独立的个人，而不是一个组织或机构 |
| 强调交互体验 | 要求产品具有低的使用成本和学习成本，重视交互设计 |
| 数据驱动设计 | 对每一个按钮、组件、页面进行全面、精确的数据监控 |
| 收益容易量化 | 核心指标包括日活、UV、PV、转化率等，项目收益容易量化和评判 |
| 运营决定存亡 | 产品运营和产品设计同样重要 |

**按设备分类**
- **PC端产品**：Web版官网、主站，PC软件（如迅雷、360助手）
- **移动端产品**：原生App、小程序或H5应用
- **其他设备端产品**：智能硬件、车载环境下运行的软件

**按功能分类**
- **工具类产品**：提供独立功能解决某一类具体需求（如墨迹天气、美图秀秀）
- **内容类产品**：为原创或聚合内容提供分享平台（如今日头条、喜马拉雅FM）
- **社交类产品**：实现陌生人、熟人之间的沟通交流（如微信、脉脉）
- **平台类产品**：作为双边市场平台服务方，帮助买卖双方实现交易撮合（如淘宝、滴滴、Airbnb）

### 1.2.2 B端产品（2B产品）

**定义**
- B端产品也叫2B(to Business)产品
- 使用对象是企业或组织
- 用来解决某类经营管理的难题

**B端产品分类**

| 分类依据 | 类型 |
|----------|------|
| 按产品特点 | 业务型产品、工具型产品、交易型产品、基础服务产品 |
| 按交付方式 | 乙方外采（定制化项目制、SaaS产品）、甲方自研自用 |

**详细分类**
- **业务型产品**：ERP、CRM等，解决企业整体性问题
- **工具型产品**：电子签章、会议系统等，解决局部明确问题
- **交易型产品**：交易系统，支撑企业商品服务售卖与履约
- **基础服务产品**：权限管理、消息推送等公共组件和服务

### 1.2.3 中台产品

**概念**
- 中台本身是一种软件抽象设计的思想，已经在IT领域存在了很多年
- 中台产品并没有权威、统一的定义
- 可能包括：电商交易系统模块（订单、商品）、统一定价中心、内容分发中心、数据中台等

**中台产品经理**
- 是B端产品经理的子集
- 工作范围各不相同，具体从事方向要看企业的具体定义

### 1.2.4 SaaS产品

**技术角度**
- SaaS是指软件的部署、交付方式
- 将标准化软件产品部署在云端
- 通过多租户形式提供给不同的客户使用

**与PaaS、IaaS的关系**

| 类型 | 说明 |
|------|------|
| SaaS | 甲方用户直接使用终端软件，但不用关心背后的基础设施 |
| PaaS | 甲方用户通过第三方提供的环境开发、维护、部署自己的软件系统 |
| IaaS | 甲方用户使用第三方提供的计算、存储能力，包括操作系统、中间件、服务器等 |

### 1.2.5 数据与策略产品

**定义**
- 对企业内外部所有数据进行挖掘并利用的产品
- 通过数据反映出来的深层次信息来有效提升对应业务的绩效

**工作方向**
- **数据仓库建设**：设计公司底层数据仓库，包括业务数据模型、指标体系等
- **应用层设计**：报表、BI等可视化工具
- **算法策略输出**：画像标签建设、策略算法（如推荐策略、派单路径规划）
- **数据监控**：设计并持续优化监控工具

**使用对象分类**
- **对内产品**：供企业内部使用，给企业赋能
- **对外产品**：将企业的数据、算法或报表工具提供给外部客户使用

### 1.2.6 商业变现产品

**定义**
- 帮助互联网企业将流量转化为收入的产品

**主要类型**
- **搜索引擎营销(SEM)**：竞价排名，按点击付费(CPC)
- **广告投放平台**：设计创意广告形式及投放管理后台
- **在线广告联盟(Ad Network)**：广告主发布广告内容，中小网站/App承接广告
- **增值服务**：给消费者提供的比基础服务更高层次的服务（如QQ虚拟道具、会员权益）

### 1.2.7 AI产品

**定义**
- 利用人工智能技术的产品
- AI产品体系包罗万象：操作系统、图像识别、语音交互、无人驾驶等

**核心要素**
- 数据、算法和应用场景的结合

**功能模块**
- 数据的收集处理
- 算法的优化
- 应用场景的设计和策略的落地

**典型应用方向**
- 图片自动鉴黄
- IVR语音交互（如车险热线机器人对话）
- 智能呼叫中心语音应答
