## 📝 如何使用

### 1. 浏览学习

通过左侧文件列表，按知识模块系统学习。

### 2. 搜索查找

使用 Obsidian 搜索功能（`Ctrl/Cmd + P`）快速定位内容。

### 3. 图谱浏览

打开关系图谱，查看知识点之间的关联。

### 4. 使用模板

在 `99-模板与工具/` 中找到工作模板，复制使用。

### 5. 双向链接

点击文档中的 `[[链接]]` 跳转到相关知识。
