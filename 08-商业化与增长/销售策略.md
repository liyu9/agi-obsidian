# 销售策略

## 销售组织

### 直销
- **电销**：承接小客户，客单价低，远程销售降低成本
- **面销**：承接大客户，面谈增进客情，提高赢单概率

**SaaS时代要求**：
- 定位清晰、价值突出的标准化产品
- 不过度依赖销售人员个人能力

### 渠道销售
- 授权其他公司代理销售
- 核心问题：渠道商需要SaaS化销售转型
- 做好渠道的关键：练好内功，才能对外赋能

## 销售流程四阶段

### 1. 线索阶段
- 客户需求不明确，无预算
- 做好打持久战准备
- 注意前期不要投入过多资源

### 2. 商机阶段

**判断条件**：
1. 需求是否明确具体，可书面记录
2. 是否有预算
3. 是否有决策流程
4. 是否有进度计划

### 3. 新签阶段
- 得到客户承诺后签署合同
- 销售人员做好交接工作（人的交接 + 事的交接）
- 维系客情促进续约

### 4. 续签阶段
- 第一年可由销售经理负责
- 后续转交客户成功经理

## CRM 系统价值

| 价值 | 说明 |
|------|------|
| 统一管理 | 客户资源不因人员流动丢失 |
| 标准化 | 销售流程和动作标准化 |
| 自动化 | 线索自动分配、提醒等 |
| 数据化 | 客观评价绩效，指导培训 |

## 产品与销售协同

- 以标准化产品为核心，不被个别客户牵着鼻子走
- 产品经理了解销售工作，帮助完成KPI
- 相互理解，携手推进产品增长
