## 五、备选方案

### 方案B：最小改动方案（仅整理根目录）

如果不想大规模重组，只整理根目录散落文件：

```
调整内容：
1. 创建"飞书集成/"目录 → 移入飞书相关文档
2. 创建"学习笔记/"目录 → 移入课程相关文档
3. 创建"待处理/"目录 → 移入临时文件
4. 其他目录结构保持不变
```

**优点：** 改动小，风险低
**缺点：** 无法完全对齐知识图谱框架

## 六、风险评估

| 风险 | 影响 | 缓解措施 |
|------|------|----------|
| 文件移动后链接失效 | Obsidian 内部链接断裂 | 移动前备份，移动后批量替换 |
| 路径过长 | Windows 路径260字符限制 | 使用较短目录名 |
| 误删重要文件 | 数据丢失 | 先备份再操作 |
| Obsidian 索引失效 | 搜索功能异常 | 移动后重建索引 |

**建议：** 操作前先关闭 Obsidian，使用文件管理器移动，完成后重新打开 Obsidian。
