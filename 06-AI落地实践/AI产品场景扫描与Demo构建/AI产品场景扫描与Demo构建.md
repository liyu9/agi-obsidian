## 构建产品小样（D帆）

### D帆定义
产品Demo = 能跑起来的"死"产品

### 构建工具

| 工具 | 说明 |
|------|------|
| Dify | 拖拽式工作流编排平台 |
| Coze | 字节跳动出品 |
| FastGPT | 国产开源 |

### Dify核心概念

| 概念 | 说明 |
|------|------|
| 应用 | 一个完整的AI功能模块 |
| Workflow | 工作流编排 |
| 知识库 | RAG知识库 |

### Dify使用步骤

1. **创建应用**：定义AI功能
2. **构建Workflow**：拖拽节点编排流程
3. **发布测试**：部署上线验证效果

## AI落地实践案例

### 案例一：家电消费品使用问答助手

**用户痛点：**
- 热水器故障 → 找不到说明书 → 求助客服
- 说明书被扔掉是常态

**企业痛点：**
- 80%问题是产品如何使用的基本问题
- 雇佣10名全职客服，流动性高
- 长尾诉求分析耗时

**产品设计：**
- RAG + 知识库问答
- 长尾诉求转人工 + AI辅助审核

### 案例二：企业级AI搜索

**核心能力：**
- Query重写
- 意图识别
- 路由分发

### 案例三：智能工单助手

**产品目标：**
- 解答问题
- 安排上门维修

**技术架构：**
- 意图识别 → TicketCreateAPI
- RAG知识库
- Re-Act模式

### 案例四：自助餐门店助手

**应用场景：**
- 帮门店回答问题
- 进行验假货、查返修

**核心方法：**
- Dify搭建MVP
- 验证市场需求

## 关键总结

| 步骤 | 说明 |
|------|------|
